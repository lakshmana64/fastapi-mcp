# Roadmap

This roadmap is a living plan and may evolve based on community feedback.

## Current Focus (Next 1-2 releases)

- Strengthen operational defaults and production presets
- Expand examples for common agent/client integrations
- Improve observability docs (metrics, traces, alerts)
- Add compatibility test matrix across client versions

## Near-Term

- Optional OpenTelemetry exporters
- Policy-driven allow/deny tool execution hooks
- More transport/hosting deployment guides
- Optional stricter argument validation mode

## Longer-Term

- Plugin interface for custom security modules
- Extended MCP resources/prompts coverage (if aligned with project scope)
- Performance benchmarking suite and published results

## How to Influence Priorities

Open a GitHub issue with:

- Use case
- Proposed behavior
- Why this should be prioritized
